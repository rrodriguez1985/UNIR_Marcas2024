Listas en HTML5

Las listas en HTML5, nos permiten organizar el texto mediante un formato más elegible y conciso. En HTML5, podemos encontrarnos/utilizar tres tipos de listas:
  • Listas Ordenadas (ol): Las listas ordenadas, vienen de OrderedList y se representan
    con la etiqueta de apertura y cierre ol
  • Listas Desordenadas (No Ordenadas): Las listas desordenadas, vienen de UnorderedList,
    y se representan con la etiqueta de apertura y cierre ul
  • Listas de Definicion (dl): Las listas de definición, vienen dadas de Definition List, y
    tienen la etiqueta de apertura y de cierre dl

Muy importante: dentro de una lista, sea del tipo que sea, podemos insetar cualquier tipo de información: imágenes, vídeis, tablas,
textos, etc.

Listas Ordenadas (ol)

La estructura de una lista ordenada (ol), viene dada por la etiqueta de apertura y cierre ol Su estructura es la siguiente:
  • Etiqueta de apertura ol
  • Posteriormente, utilizaremos la etiqueta li, que tiene etiqueta de apertura y cierre.
    Esta etiqueta es donde insertaremos el texto o contenido de nuestra lista.
  • Por último, tendríamos la etiqueta de cierre ol

Listas Desordenadas (ul)

La estructura de una lista desordenada (ul), viene dada por la etiqueta de apertura y cierre ul Su estructura es la siguiente:
  • Etiqueta de apertura ul
  • Posteriormente, utilizaremos la etiqueta li, que tiene etiqueta de apertura y cierre.
    Esta etiqueta es donde insertaremos el texto o contenido de nuestra lista.
  • Por último, tendríamos la etiqueta de cierre ul

Atributos asociados a las Listas Ordenadas y Desordenadas

Tanto en las listas ordenadas como desordenas, podemos utilizar una serie de atributos que nos van a permitir
modificar/customizar, nuestras listas.
En el caso de las listas ordenadas, podemos modificar el orden de comienzo, así como el tipo.
Para ello, utilizaremos el atributo start para indicar el orden de comienzo, y el atributo
type para indicar el tipo de forma que queremos aplicar.
En el caso de las listas desordenadas, sólo podemos modificar el tipo, aplicando en este caso también
el atributo type y modifican el icono gráfico de este.
  • Para listas Ordenadas: Aa para indicar que empiece con las letras del abecedario
  • Para listas Ordenadas:Ii para indicar que empiece con números romanos
  • Para listas Ordenadas:Números predeterminados (1,2,3...)
  • Para listas desordenadas: disc (viñeta redondeada rellena)
  • Para listas desordenadas: circle (viñeta redondeada vacía)
  • Para listas desordenadas: square (viñeta cuadra rellena)

Listas de Definición (dl)

Las listas de definición, son utilizadas como bien dice su nombre, para poder crear un documento en el que
tengamos que realizar definiciones de términos, crear una wiki, un diccionario, etc.
Al contrario de las listas ordenadas y desordenadas, las etiquetas y estructura es distinta.
  • dl: Etiqueta de apertura y cierre dl. Description List y que sirte
    para delimitar la lista completa
  • dt: Viene dada de Description Term, y delimita los términos de nuestra lista
  • dd: Viene dada de Definition Description, y delimita las definiciones de nuestra lista

Hipervínculos o Hipertexto en HTML5

Los hipertextos o hipervínculos, son un elemento característico de HTML5 que nos permite que las páginas web puedan relacionarse entre sí.
Los hipervínculos, están formadas por dos partes:
  • El enlace: es el elemento (pág. web, imagen, archivo, etc.) sobre el que se puede hacer click con el ratón.
  • El destino: El elemento (pág. web, imagen, archivo, etc.), que se muestra al hacer click en el enlace.

Los hipervínculos en HTML5 tienen la siguiente estructura interna:
  • La etiqueta de apetura y cierre a (anchor).
  • El atributo href, es dónde vamos a poner el link del elemento que queremos como destino.
  • Entre la etiqueta de apertura y la etiqueta de cirre de a, el texto del hipervínculo o hiperenlace
